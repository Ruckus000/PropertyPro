## TASK 1: P0-06 — Scoped Query Builder (CRITICAL SECURITY BOUNDARY)

This is the most important task in Phase 0. It prevents cross-tenant data leaks.

### Files to Create

All in `packages/db/src/`:

**`tenant-context.ts`** — Tenant context management:
```ts
// Stores the current community context
// Used by scoped client to auto-inject community_id filters
export interface TenantContext {
  communityId: bigint;
}
```

**`errors/TenantContextMissing.ts`**:
```ts
export class TenantContextMissing extends Error {
  constructor() {
    super('Tenant context is required but was not provided. All queries must include a community_id.');
    this.name = 'TenantContextMissing';
  }
}
```

**`scoped-client.ts`** — The core security layer:

This must implement `createScopedClient(communityId: bigint)` that returns a wrapper around Drizzle with these behaviors:

1. **Auto-inject community_id:** Every SELECT, UPDATE, DELETE query automatically gets `.where(eq(table.communityId, ctx.communityId))` appended
2. **Auto-inject soft-delete filter:** Every query on soft-deletable tables gets `.where(isNull(table.deletedAt))` appended
3. **Audit log exception [AGENTS #8]:** `compliance_audit_log` table is EXEMPT from soft-delete filtering (append-only, never deleted)
4. **Throw on missing context:** If communityId is undefined/null, throw `TenantContextMissing` (NOT empty results)
5. **Override explicit bypass attempts:** If a caller tries to pass a different communityId in their WHERE clause, the scoped client MUST override it with the context's communityId

The scoped client should provide these methods at minimum:
- `query(table)` — SELECT with auto-scoping
- `insert(table, data)` — INSERT with auto-injected community_id
- `update(table, data, where)` — UPDATE with auto-scoping
- `softDelete(table, where)` — Sets deleted_at = now() with auto-scoping
- `hardDelete(table, where)` — Actual DELETE (use sparingly, only for non-tenant data)

**CRITICAL:** The raw `db` object from drizzle.ts must NOT be exported from packages/db. Only `createScopedClient` should be the public API for database access. [AGENTS #14]

## TASK 2: P0-07 — Error Handling

### Files to Create

All in `apps/web/src/`:

**`lib/api/errors/AppError.ts`** — Base error class:
```ts
export class AppError extends Error {
  constructor(
    message: string,
    public readonly statusCode: number,
    public readonly code: string,
    public readonly details?: Record<string, unknown>
  ) {
    super(message);
    this.name = 'AppError';
  }

  toJSON() {
    return {
      error: {
        code: this.code,
        message: this.message,
        ...(this.details && { details: this.details }),
      },
    };
  }
}
```

**`lib/api/errors/ValidationError.ts`** — 400:
```ts
export class ValidationError extends AppError {
  constructor(message: string, details?: Record<string, unknown>) {

    this.name = 'ValidationError';
  }
}
```

**`lib/api/error-handler.ts`** — withErrorHandler HOF:
```ts
import { NextResponse, type NextRequest } from 'next/server';
import { AppError } from './errors/AppError';
// Import Sentry when P0-08 is done

type RouteHandler = (
  req: NextRequest,
  context?: { params: Record<string, string> }
) => Promise<NextResponse>;

export function withErrorHandler(handler: RouteHandler): RouteHandler {
  return async (req, context) => {
    try {
      const response = await handler(req, context);
      return response;
    } catch (error) {
      if (error instanceof AppError) {
        return NextResponse.json(error.toJSON(), {
          status: error.statusCode,
          headers: { 'X-Request-ID': req.headers.get('x-request-id') ?? '' },
        });
      }

      // Unknown error — 500, no stack trace exposed [AGENTS #43]
      console.error('Unhandled error:', error);
      // TODO: Sentry.captureException(error) after P0-08

      return NextResponse.json(
        { error: { code: 'INTERNAL_ERROR', message: 'An unexpected error occurred' } },
        {
          status: 500,
          headers: { 'X-Request-ID': req.headers.get('x-request-id') ?? '' },
        }
      );
    }
  };
}
```

```ts
Sentry.init({
  dsn: process.env.SENTRY_DSN,
  enabled: !!process.env.SENTRY_DSN,
  tracesSampleRate: process.env.NODE_ENV === 'production' ? 0.1 : 1.0,
  beforeSend(event) {
    // Redact sensitive headers [acceptance criteria]
    if (event.request?.headers) {
      delete event.request.headers['authorization'];
      delete event.request.headers['cookie'];
      delete event.request.headers['x-api-key'];
    }
    return event;
  },
});
```
